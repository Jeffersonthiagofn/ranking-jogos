# Ranking de Jogos Steam / Steam Games Ranking

[🇧🇷 Leia em Português](./README.pt-BR.md) | [🇺🇸 Read in English](./README.en.md)

Aplicação full stack para ranking e análise de jogos da Steam.  
Full-stack application for Steam game ranking and analysis.

---

## Contexto acadêmico / Academic context

### 🇧🇷 Português

Este projeto foi desenvolvido na **UNIFACISA** como parte das disciplinas e competências:

- **Integrar Interfaces e Serviços Web**
- **Usabilidade UI/UX**

O processo de desenvolvimento envolveu duas etapas principais:

1. **Protótipação da interface no Figma**, com foco em experiência do usuário, organização da navegação e definição das telas;
2. **Implementação do sistema em código**, transformando o protótipo em uma aplicação funcional full stack.

### 🇺🇸 English

This project was developed at **UNIFACISA** as part of the following courses and competency areas:

- **Integrating Interfaces and Web Services**
- **UI/UX Usability**

The development process involved two main stages:

1. **Interface prototyping in Figma**, focusing on user experience, navigation structure, and screen design;
2. **System implementation in code**, turning the prototype into a functional full-stack application.
